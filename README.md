# Auto-resolve Jira Tickets with GitHub Copilot using Port Context

Advanced n8n automation for Auto-resolve Jira Tickets with GitHub Copilot using Port Context.

## Overview
- Category: Ticket Management, AI RAG
- Complexity: intermediate
- Source: n8n workflow template export

## What This Automation Does
Automate Jira to GitHub workflows with n8n, Port, and Copilotpreserve rich context, enhance traceability, and speed up development. Learn more!

## Included Files
- `workflow.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.

## Tech Stack

- `CUSTOM.portIo`
- `n8n-nodes-base.github`
- `n8n-nodes-base.if`
- `n8n-nodes-base.jira`
- `n8n-nodes-base.jiraTrigger`
- `n8n-nodes-base.stickyNote`
